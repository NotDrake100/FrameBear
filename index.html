⟩ What It Does
🎯

Prompt to Video

Describe what you want. Add a reference video. Get a finished MP4 in seconds. The AI handles the creative direction.

🧠

Any AI Model

Gemini, ChatGPT, Claude, or local models via Ollama. Bring your own API key — or use a free tier.

🔒

Runs Locally

Everything runs on your machine. Your prompts, videos, and brand assets never leave your computer.

🎬

Reference Videos

Upload any product video as reference. The AI analyzes the style, timing, and transitions — then recreates it for your brand.

⚡

Frame-Perfect

Virtual time system captures every frame deterministically. No dropped frames, no timing issues. Same result every time.

🎵

Audio Support

Drop in background music. Auto-synced and encoded as AAC. Production-ready output in H.264 MP4.

⟩ Quick Start

Up and running in 30 seconds

Clone the repo, add your API key, start generating videos.

git clone https://github.com/NotDrake100/FrameBear.git && cd FrameBear && npm i

Then run: npx framebear init to connect your AI model. Requires Node.js 18+, Chrome & FFmpeg.
